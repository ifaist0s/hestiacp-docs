Getting Started
==================

Welcome! 

Hestia is an open source `web hosting control panel <https://en.wikipedia.org/wiki/Web_hosting_control_panel>`_ for Linux servers.
---------------------------
Supported operating systems
---------------------------
Hestia can be installed on:

* Debian 8
* Debian 9
* Ubuntu 16.04 LTS
* Ubuntu 18.04 LTS
Debian 10 will be supported with the upcoming version of Hestia 1.1.0.

Prerequisites
-------------
Installing and maintaining Hestia requires basic skills with the Linux command line and server management.

  Current documentation is subject of change. Hestia is under active development now.
